# opt-device-simulator

## Feature
- 修改模块名和命名空间（已完成）
- 能力集根据依赖的yang model自动注入
- 删除resource/yang目录，根据依赖的yang model生成对象树
- 支持10种类型模拟器，与网管DC模块对应
- 支持自动化打包部署
- 提供控制器，用于监视和管理设备模拟器



## Feature

- 支持通过页面的方式，通过配置IP，端口范围，设备类型的方式进行部署不同设备类型的设备模拟器
- 支持通过页面对配置的设备模拟器进行启动，停止，重新配置等操作
- 支持通过页面选择设备模拟器来进行告警数据模拟，并控制其发送条数和发送频率的方式来进行数据模拟
- 支持同时启动多个不同设备类型，不同设备版本的设备模拟器
